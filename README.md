# Finance Dashboard

A clean and interactive finance dashboard built using React. It allows users to track transactions, view financial summaries, and understand spending patterns through simple visualizations.

---

## Features

* Financial summary including total balance, income, and expenses
* Transaction management (add, edit, delete)
* Filtering by type and category
* Role-based UI (Admin / Viewer)
* Insights such as highest spending category and savings rate
* Dark mode support
* Data persistence using localStorage
* Smooth UI with basic animations

---

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Framer Motion

---

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/Engamansinghtomar/finance-dashboard.git
```

2. Navigate to the project folder

```bash
cd finance-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

---

## Approach

The project is structured using reusable components and centralized state management with Context API. The focus was on building a simple, intuitive interface while maintaining clean code and modularity.
